# AV-Plan Trafikkskilt

## Om disse skiltene

Dette er enkle placeholder-skilt for utvikling og testing av AV-Plan. De er designet for å være gjenkjennelige og funksjonelle, men er ikke offisielle Statens vegvesen-skilt.

## Skilttyper

### Fartsgrenser (speed/)
- 30.svg - 30 km/t
- 40.svg - 40 km/t
- 50.svg - 50 km/t
- 60.svg - 60 km/t
- 70.svg - 70 km/t
- 80.svg - 80 km/t
- 90.svg - 90 km/t
- 110.svg - 110 km/t

### Varslingsskilt (warning/)
- 142.svg - Arbeid på vegen
- 204.svg - Farlig vegkant
- 110.svg - Ukjent fare

### Forbudsskilt (prohibition/)
- 306.svg - Forbikjøring forbudt
- 308.svg - Innkjøring forbudt

## Erstatte med offisielle skilt

For produksjonsbruk, erstatt disse skiltene med offisielle SVG-filer fra:
- Statens vegvesen sin skilthåndbok
- Eller andre offisielle kilder

Sørg for at filnavnene og mappestrukturen forblir den samme, slik at `data/sign-library.json` fortsatt fungerer.

## Format

Alle skilt er i SVG-format med:
- ViewBox: 0 0 100 100
- Konsistent styling
- Høy kontrast for synlighet
- Skalerbar vektorgrafikk

## Lisens

Disse placeholder-skiltene er kun for utvikling. For produksjon, bruk offisielle skilt med korrekt lisens.
