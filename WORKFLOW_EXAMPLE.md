## Pro Tips

### Efficient Workflow
1. **Search first** - Always start with address search
2. **Zoom before export** - Set ideal view before exporting
3. **Save often** - Save after major changes
4. **Name clearly** - Use descriptive project names with dates
5. **Test loading** - Always test loading saved projects

### Sign Placement Best Practices
1. **Advance warning** - Place warning signs 200m before work
2. **Speed reduction** - Gradual speed limit changes
3. **Visual clarity** - Don't overlap signs
4. **Direction matters** - Rotate signs to face traffic
5. **Both directions** - Consider traffic from both ways

### Quality Checks Before Export
- [ ] Work zone fully visible in view
- [ ] All signs readable
- [ ] Distance markers present
- [ ] Scale bar will be included
- [ ] Zoom level appropriate

## Common Scenarios

### Short Work Zone (< 100m)
```
- Fewer distance markers
- Closer sign spacing
- 50m warnings sufficient
```

### Long Work Zone (> 500m)
```
- Full 400m marker ranges
- Multiple warning signs
- Consider sections
```

### Complex Intersection Work
```
- Multiple road selections
- Additional warning signs
- More careful placement
```

### Emergency/Temporary Work
```
- Quick placement
- Essential signs only
- Fast export
- Update as needed
```
